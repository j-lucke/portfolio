USMNT social

See who's trending on the US Men's National Team with USMNT-social. Launched in January 2023 (just after the world cup) this site uses the twitter API to track followers and mentions for everybody whose anybody in the world of US Men's soccer. Data is culled using hourly chron jobs that run on Node.JS and write to a Postgres Data base.
